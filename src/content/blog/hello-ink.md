我一直想把「东方的克制」和「现代 Web 的交互」缝合在一起。这个站点是一次尝试：用 WebGL 流体模拟在浏览器里重现毛笔在宣纸上的晕染。

## 为什么是水墨

代码是绝对理性的 0 与 1，但呈现可以有呼吸感。留白、晕染、朱砂一点，都是节奏。

## 技术上做了什么

- 用 three.js 跑一个简化的流体场，跟随光标推进墨迹。
- 用 GSAP + ScrollTrigger 让内容像卷轴一样徐徐展开。
- 用 Astro 的 View Transitions 做页面之间的水墨过场。

```ts
// 墨迹推进的核心：涡度 + 平流
vel += vec2(curl, -curl) * 0.4;
```
