## 六仪击刑

六仪击刑即所谓的六个天干——六仪遇到了相刑的地支形成击刑。凡是出现了击刑办事说明凶兆，办事是很不利的，好事会出问题，坏事问题更大。

|   | 巳        | 午   | 未   |   |
| - | -------- | --- | --- | - |
| 辰 | 壬六击  癸六击 | 辛六击 | 己六击 | 申 |
| 卯 | 戊六击      |     |     | 酉 |
| 寅 | 庚六击      |     |     | 戌 |
|   | 丑        | 子   | 亥   |   |

### 庚六击

甲申庚，申元帅躲在此处，申与寅产生寅申刑，冲击之刑。

### 戊六击

甲子戊，子卯产生子卯刑，为无礼之刑。

### 壬六击，癸六击

甲辰壬，辰辰自相刑法，叫辰辰自刑，内斗。

甲寅癸，寅巳刑，寅为阳木，巳为阴火，阳生阴不合道理，不能让阳性去生阴性。

### 辛六击

甲午辛，午午自刑。

### 己六击

甲戌己，未戌刑。

> 一般分析击刑不将击刑写在宫内，写在宫的旁边进行分析使用。

## 入墓

入墓只有在四个角上才会有。

在左下角艮土宫为丁入墓、己入墓、庚入墓。

在左上角巽木宫为辛入墓、壬入墓。

在右上角坤土宫为乙入墓（原本是甲，隐藏需要使用乙代替）、癸入墓。

在右下角乾水宫为乙入墓、丙入墓、戊入墓。

## 空亡

空亡通过查询六十甲子表都能查询出来，表示这个事情不落实，只有将这个空填补（填空）或者冲掉（冲空）才能落实。分析的时候需要将空在格子旁边标注出来，否则容易漏掉。

## 马星

马星是每个时辰最活跃的冲动力量。可以冲掉入墓，可以冲掉空亡，将不变变为变动。

### 不同时辰的马星

- 寅午戌时：马星是要对着冲的，所以马星在申（寅申冲）。然后寅午戌他们三合形成一种力量，所以他们三个时辰的马星都在申。
- 巳酉丑时：同理，领头的是巳，则马星为亥（亥巳冲）。然后巳酉丑他们三合形成一种力量，所以他们三个时辰的马星都在亥。
- 申子辰时：马星为寅（寅申冲）。
- 亥卯未时：马星为巳（亥巳冲）。

## 实际案例

![六仪击刑、入墓、空亡和马星·图1](/blog/liuyi-jixing/1.png)

先分析六击，首先艮土宫，如果有庚了就发生了击刑，上面的图有庚，所以需要在格子旁边标注庚六击。

其次分析入墓，艮土宫如果丁己庚就发生了入墓，现在有了庚，所以需要在格子旁边标注庚入墓。坤土宫如果有乙癸就发生入墓，现在有了乙癸，所以乙入墓和癸入墓。乾水宫如果有乙丙戊则发生入墓，现在有了戊，则戊入墓。

然后分析日空和时空，日为戌亥空，则戌空，亥空，标注到右下格子旁边。时为辰巳空，则辰空，巳空，标注到左上格子旁边。

最后标注马星，马星相应于时辰，现在是未时，未时对应亥卯未，对着亥冲过去为巳，则马星在巳位。
