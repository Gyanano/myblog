> 九星指的是外部的宇宙因素，八神指的是外部的力量，可能是你已知的力量，也可能是未知的力量，这两种参数需要分辨清楚。

八神分别是：

- 值符（符）
- 螣蛇（蛇）
- 太阴（阴）
- 六合（合）
- 白虎（虎）
- 玄武（武）
- 九地（地）
- 九天（天）

值符是八神的元首，同时也是九星的领军人物，值符每次去执行任务都需要带一颗九星。

## 八神的类象

### 值符：中央土

贵人|原告|债主|大集团|主管领导|防守者

八神九星的领袖 所到之处百灾俱消 天乙之神

### 螣蛇：南方火

纠缠力量丨身材苗条|噩梦

虚诈之神 为人虚伪狠毒 有纠缠力量 怪异的现象

### 太阴：西方阴金

暗中荫护 保护神 有隐匿性格 显而不露

**临宫时**，利于出谋划策 避凶 藏兵

### 六合：东方木

恋爱婚姻|合作|中介|证据|逃犯

多方面保护神 为人开朗平和 人际关系良好 利于合作 利于中介 利于谈判 利于交易 利于婚娶

### 白虎：西方金

武警|血光之灾|行凶打人

凶煞之神 为人凶狠好斗 牵涉行军打仗 凶杀格斗 疾病伤亡 交通事故

### 玄武：北方水

小偷|迷糊丨假品丨吹嘘丨夸大宣传

虚伪的小偷小摸的神 喜欢偷窃 喜欢偷情 喜欢弄虚作假 易发生迷糊

### 九地：坤土

低矮陈旧|农作物

厚的坚牢的神 温顺安静 宽厚让人

**临宫时**，利于屯兵固守 跟踪 养殖

### 九天：乾金

高档|农作物「新颖丨航天|航空|航线

乾金宫威严之神 刚健好动 自强不息

**临宫时**，可以扬兵 征战 可以远行

## 八神的排序

八神在九宫中无固定位置，神无处不在。但是神之间是有排序的，即“符蛇阴合虎武地天”。

同时八神的排序遵循阳顺阴逆，而八门和九星却始终按照顺时针的顺序排序。

> 通常八神喜欢写在每个宫格的左上端。

### 阳遁局

假设值符在右上角位置，即坤土宫。即他们的排序如下：

| 地 | 天 | 符 |
| - | - | - |
| 武 |   | 蛇 |
| 虎 | 合 | 阴 |

### 阴遁局

假设值符在中下的位置，即坎水宫。即他们的排序如下：

| 武 | 虎 | 合 |
| - | - | - |
| 地 |   | 阴 |
| 天 | 符 | 蛇 |
